Module 1: كلام مباشر مع المستخدم (Dialog Boxes)

إيه يا صاوى! عايز أخلي الصفحة تطلع رسالة للمستخدم، أو تسأله عن اسمه. أعملها إزاي دي؟

دي سهلة يا جمعة، عندك 3 صناديق حوار جاهزين تحت أمرك.

الأول alert()، ده بتاع "خد بالك!" أو "تم بنجاح". مجرد رسالة وزرار OK.

alert("تم حفظ البيانات!");

التاني confirm()، ده بتاع الأسئلة اللي إجابتها أه أو لأ. بيرجعلك true لو داس OK و false لو داس Cancel.

let wantsToDelete = confirm("متأكد عايز تمسح؟");

والتالت prompt()، ده اللي بيخليك تاخد كلام من المستخدم. بيرجعلك الكلام اللي كتبه.

let name = prompt("اكتب اسمك يا بطل:");

بس خلي بالك، الحاجات دي قديمة وبتوقف الصفحة كلها لحد ما المستخدم يرد. في الشغل بنعمل أشكال أحسن بالـ HTML والـ CSS.

جرب بنفسك

Module 2: مصنع الأكواد (Functions)

يا صاوى، أنا بكتب كود كتير بيتكرر. عايز أعمل حاجة زي "مصنع" صغير، أديله شوية حاجات ويطلعلي نتيجة. إيه الحل؟

الحل هو الـ Functions يا جمعة. دي بتخليك تكتب الكود مرة واحدة وتستخدمه 100 مرة. بص كده:

// 1. بنبني المصنع وبنقوله هياخد إيه (parameters)
function createGreeting(name) {
  const greeting = "أهلاً بيك يا " + name;
  return greeting; // 2. بنقوله هيرجع إيه
}

// 3. بنشغل المصنع ونديله الحاجات اللي عايزها (arguments)
let message = createGreeting("جمعة");
console.log(message); // "أهلاً بيك يا جمعة"

Module 3: التكرار يعلم الشطار (Loops)

طب لو عايز أطبع الأرقام من 1 لـ 5؟ مش معقول هكتب console.log خمس مرات!

أكيد لأ. هنا يجي دور الـ Loops. لو عارف عدد المرات، استخدم for loop.

// (البداية; الشرط اللي هيوقفنا; الخطوة اللي بنمشيها)
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
// 1, 2, 3, 4, 5

طب لو عايز أفضل أسأل عن الباسورد لحد ما يتكتب صح؟ أنا مش عارف هياخده من كام مرة!

في الحالة دي، استخدم do...while. دي بتنفذ الكود مرة واحدة على الأقل، وبعدين تفضل تكرر طول ما الشرط متحقق.

let password;
do {
  password = prompt("Enter the password:");
} while (password !== "secret");

alert("Access granted!");

Module 4: التعامل مع النصوص (String Object)

يا صاوى، عايز ألعب في الكلام. عندي جملة وعايز أعرف طولها، وأشيل المسافات الزيادة، وأخليها كلها حروف كبيرة!

ده شغل الـ String Object. أي نص عندك بيكون معاه شنطة مليانة أدوات (methods) جاهزة. بص كده:

let text = "  JavaScript is Fun!  ";

// .length بيجيب الطول
console.log(text.length); // 23

// .trim() بيشيل المسافات من الأول والآخر
let cleanText = text.trim(); 
console.log(cleanText); // "JavaScript is Fun!"

// .toUpperCase() بيخلي كله كابيتال
console.log(cleanText.toUpperCase());

// .split() بيكسر النص ويحوله لـ array
let words = cleanText.split(' ');
console.log(words); // ["JavaScript", "is", "Fun!"]

Module 5: لمّة العيلة (Array Object)

يا صاوى، عندي قايمة درجات وعايز أعمل عليها عمليات. عايز أطلع قايمة جديدة بالدرجات الناجحة بس (أكبر من 50).

ده شغل الـ Array Object وأدواته السحرية. بدل ما تعمل loop بنفسك، استخدم .filter().

let scores = [80, 45, 90, 30, 65];

// .filter() بتعدي على كل عنصر وتشوف الشرط
// لو الشرط صح، بتحط العنصر في الأراي الجديدة
let passingScores = scores.filter(function(score) {
  return score > 50;
});

console.log(passingScores); // [80, 90, 65]

يا سلام! طب لو عايز أعمل قايمة جديدة فيها كل الدرجات بعد ما أزود 5 درجات بونص لكل طالب؟

هنا تستخدم .map(). دي بتلف على كل عنصر وتعمل عليه عملية وترجعلك array جديدة بالنتائج.

let scores = [80, 45, 90];

// .map() بتاخد كل درجة وتطبق عليها المعادلة
let scoresWithBonus = scores.map(function(score) {
  return score + 5;
});

console.log(scoresWithBonus); // [85, 50, 95]

Module 6: عبقري الرياضيات (Math Object)

عايز أعمل لعبة تخمين أرقام. إزاي أخلّي الكمبيوتر يختار رقم عشوائي من 1 لـ 9؟

هنا تستدعي العبقري Math. ده object جاهز فيه كل العمليات الحسابية اللي تتخيلها. عشان تجيب رقم عشوائي، بنعمل تركيبة بسيطة:

// Math.random() بتجيب رقم عشري من 0 لـ 0.999
// بنضربه في 9 عشان يبقى من 0 لـ 8.999
// Math.floor() بتقص الكسر، يبقى من 0 لـ 8
// نزود 1 عشان يبقى من 1 لـ 9
const randomNumber = Math.floor(Math.random() * 9) + 1;
console.log(randomNumber); // رقم عشوائي بين 1 و 9

فهمت التركيبة. طب إيه حوار التقريب ده؟ floor و ceil و round؟

بص يا سيدي: floor بينزل للأرض (بيشيل الكسر). ceil بيطلع للسقف (بيقرّب للرقم الأكبر). round بيقرّب للأقرب.

console.log(Math.floor(5.9)); // 5 (الأرض)
console.log(Math.ceil(5.1));  // 6 (السقف)
console.log(Math.round(5.4)); // 5 (الأقرب)
console.log(Math.round(5.5)); // 6 (الأقرب)

Module 7: نتيجة الزمن (Date Object)

آخر حاجة يا صاوى، إزاي أعرف النهاردة تاريخ كام أو الساعة كام دلوقتي؟

هنا لازم تعمل object جديد من new Date(). ده بيديك object بيمثل اللحظة الحالية بالظبط.

// 1. بنجيب اللحظة الحالية
const now = new Date();
console.log(now.toDateString()); // "Sat Oct 21 2023"

// 2. بنطلع منها اللي إحنا عايزينه
const year = now.getFullYear();
const month = now.getMonth() + 1; // الشهور بتبدأ من 0، لازم نزود 1
const day = now.getDate();
const hours = now.getHours();
const minutes = now.getMinutes();

console.log(`التاريخ: ${day}/${month}/${year}`);
console.log(`الساعة: ${hours}:${minutes}`);
